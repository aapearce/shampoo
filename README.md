# SHAMPOO

**A Creative Writing & Comprehension Academy for ages 6–18**

Built with React + Vite. Deployed to GitHub Pages.

## Features
- Upload & Assess your creative writing (type or photograph)
- Generate & Brainstorm stories with AI
- Explore Literary Devices with classic examples
- Collectible badge rewards system
- Age-appropriate content for 6–8, 9–12, 12–15, 15+

## Design
Illuminate concept — Graphite/Oxford/Gold palette, Libre Baskerville + Nunito Sans.
